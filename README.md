Weather Forecast Application

Project Overview
	This is a weather forecast application developed using JavaScript, HTML, and CSS with integration of
    a Weather API. The application provides location-based weather forecasts including current conditions and extended forecasts for multiple days. It features a user-friendly, responsive interface designed with Tailwind CSS.

Features
	Search weather forecasts by city name

	Get weather forecast for the user's current location

	Dropdown menu for recently searched cities, stored locally

	Temperature unit toggle (Celsius/Fahrenheit) for today's temperature

	Custom weather alerts for extreme temperatures (e.g., above 40Â°C)

	Dynamic weather condition icons and backgrounds (e.g., sunny, rainy)

	5-day extended forecast display with temperature, wind speed, and humidity

	Real-time API error handling with user-friendly messages

	Input validation for search queries with error notifications

Technologies Used
	JavaScript

	HTML5

	CSS3 with Tailwind CSS framework

	Weather API (OpenWeatherMap)

Installation and Setup
	Clone the repository.

	Install any required dependencies (Tailwind).

	Open the index.html file in a web browser.

	Use the search input to find weather forecasts by city or allow location access to get your current weather.

Usage Instructions
	Enter a city name and press search to get the weather forecast.

	Use the dropdown to select recently searched cities.

	Toggle between Celsius and Fahrenheit units for the current day temperature.

	Observe the background change dynamically based on weather conditions.

	Check alerts for extreme temperature conditions.
